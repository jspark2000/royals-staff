## Description

성균관대학교 미식축구부 부원관리 플랫폼입니다. 현재 개발중입니다.<br>
프론트엔드 repository 링크: https://github.com/jspark2000/royals-staff-frontend

## Installation

```bash
$ npm install
```

## Running the app

```bash
# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod

# seeding database (prisma)
$ pnpm prisma migrate reset
```
